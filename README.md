# 💰 Invest Dashboard

A simple investment simulator built with JavaScript, HTML and CSS.

This application allows users to simulate financial growth over time using compound interest and monthly contributions, based on Brazilian indicators like CDI and SELIC.

## 🚀 Features

- Compound interest simulation
- Monthly contributions
- Simple and intuitive interface

## ▶️ How to run

1. Clone the repository:
2. git clone https://github.com/jotapersou/invest-dashboard.git
2. Open the project folder
3. Run the `index.html` file in your browser
